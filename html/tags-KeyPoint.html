多媒体标签

  • img: 图片标签,用于插入图片

    从技术上讲，实际上并没有将图像插入到网页中，而是将图像链接到了网页。 标签创建了一个容器，用于引用图像。
    图像的格式可以是 GIF、JPG、PNG、BMP、SVG 等。
    可被超链接标签包裹，点击图片时跳转至别的页面。标题处的GIF就是
    标签一般有两个属性：
    src - 规定图像的路径
    alt - 如果由于某种原因无法显示图像，则指定图像的替代文本
    <img>
    👉
    👉

  • audio: 音频标签,用于插入音频

    或许你一进入此页面就发现网页自动播放了音乐
    这正是audio标签的作用,tag可以播放音频文件,并可以控制播放的属性,比如循环播放、自动播放等。
    <audio> 标签可包含一个或多个拥有不同音频源的 <source>标签。浏览器将选择它支持的第一个源。
    标签一般有一个属性：
    src - 规定音频文件的路径
    但也有时候会有以下属性：
    controls - 规定是否显示播放控制按钮
    autoplay - 规定是否自动播放
    loop - 规定是否循环播放
    <audio></audio>
    👉
    👉

  • video: 视频标签,用于插入视频

    视频标签可以播放视频文件,并可以控制播放的属性,比如自动播放、循环播放等。
    <video> 标签可包含一个或多个拥有不同视频源的 <source>标签。浏览器将选择它支持的第一个源。
    标签一般有以下属性：
    src - 规定视频文件的路径
    controls - 规定是否显示播放控制按钮
    autoplay - 规定是否自动播放
    loop - 规定是否循环播放
    <video></video>

表格标签:

用于创建表格,可以插入表头、表格数据、表格样式、单元格样式等。
<table> </table> 标签用于创建表格
<tr> </tr> 标签用于创建行
<td> </td> 标签用于创建单元格
标签一般具备以下属性：
border - 规定单元格的边框宽度(单元格也能定义边框属性,如果没有定义,则会沿用表格的边框属性)
cellpadding - 规定单元格内容与单元格边框之间的空白宽度
cellspacing - 规定单元格边框与单元格内容之间的空白宽度
width - 规定表格的宽度
height - 规定表格的高度
align - 规定表格内容的水平对齐方式
valign - 规定表格内容的垂直对齐方式
bgcolor - 规定单元格的背景颜色
<table border="1px">
  <tr>
    <td>y\x</td>
    <td>1</td>
    <td>2</td>
    <td>3</td>
  </tr>

    <td>1,3</td>
  </tr>

    <td>2</td>

td标签中还有两个属性值得注意的:
跨行: rowspan="跨行行数"
跨列: colspan="跨列列数"
被跨行/列的单元格不会消失,而是被“挤到”了后面,所以在合并单元格时要格外注意,对应行/列得少n个单元格
1,1 1,2 1,3
2,1
以上为一个跨行和跨列的例子，注意第二列由于跨行和跨列，少了俩单元格。
如果没有合并单元格，则会出现空白单元格;
或者仍旧添加这俩单元格则会被“挤到”后面。
情况一:
1,1 1,2 1,3
2,1
情况二:
1,1 1,2 1,3
2,1 2,2 2,3

表单标签<form> </form>:

用于实现用户在界面上输入各种信息并将其提交的一种标签,
其中包含其他的输入标签以实现内容输入。
其一般有以下属性:
action - 规定当提交表单时向何处发送表单数据
method - 规定用于发送表单数据的 HTTP 方法(get/post)
    get - 数据添加至url后面,可通过url直接获取数据,安全性较差
    post - 数据以表单形式发送,安全性较高,但数据量较大时会受到限制

  • <input>

    用于创建文本框、密码框、单选框、复选框
    该标签一般具备以下属性:
    type - 规定输入控件的类型
    name - 规定输入控件的名称(用于提交表单中的数据、匹配单选框、复选框的选择项)
    也有其他可选的常用属性: value - 规定输入控件的默认值,而在单选框、复选框中则表示选择的选项在提交表单时提交的数据
    size - 规定输入控件的宽度(以字符计)
    disabled - 规定输入控件是否被禁用
    readonly - 规定输入控件是否是只读的
    placeholder - 规定输入控件的提示文本
    • 文本框
      其中typed属性为text,用于创建单行文本框
      <input type="text">

      👉

      其中typed属性为password,用于创建密码框
      <input type="password">
